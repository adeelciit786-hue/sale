About Sales Forecasting Dashboard

üìä What is This Dashboard?

The Enterprise Sales Forecasting Dashboard is a professional management tool designed to provide real-time sales insights and accurate monthly projections. It combines historical data analysis with intelligent forecasting to help decision-makers track progress and plan strategically.

üîç Forecasting Methodology

Weekday-Based Model

Our forecasting engine analyzes historical sales patterns by day of the week to identify consistent trends:

  • Historical Analysis: Calculates average sales for each weekday (MON‚ÄìSUN) across all available historical months
  • Current Month: Uses actual sales for completed days, applies weekday averages for forecasted future days
  • Monthly Projection: Combines actual and forecasted daily sales to project total monthly revenue

üìà Key Performance Indicators

Today's Projected Sale (AED)

Expected sales for today based on the weekday average from historical data.

Monthly Projection (AED)

Forecasted total sales for the complete month, combining actual and projected daily figures.

Monthly Target (AED)

The revenue target set for the current month (configurable in Upload Data section).

Gap (Surplus / Shortfall)

Difference between projected sales and target. Green indicates surplus, red indicates shortfall.

üíæ Data Management

Historical Data: Upload multiple months of sales data. Each file is stored independently and used for pattern analysis. Upload a file with the same name to replace it.

Current Month: Upload the current month's sales data. Only one current month file is maintained; new uploads automatically replace the previous one.

Targets: Set monthly revenue targets that persist across sessions. Targets are matched to current month files by filename.

‚ùì Frequently Asked Questions

How accurate are the forecasts?

Accuracy improves with more historical data. Minimum 2 months required; 4+ months recommended for reliable patterns.

What if data contains errors?

The dashboard automatically cleans data: dashes become zeros, non-numeric values are handled safely, and the TOTAL row is always excluded.

Can I delete files?

Yes. Deleted files are removed from analysis. The app is stable and won't crash due to file operations.

Does the dashboard work offline?

The dashboard runs locally but requires a web browser. All data is stored locally on your server.
